Es un símbolo para facilitar la automatización de personas ocupadas. Asegúrate de usarlos consistentemente.

Tipografía

La tipografía de Cuy está organizada para mantener las cosas simples, legibles y sin distracciones de la tarea en cuestión.



Tipo de letra
El tipo de letra principal de Cuy es BUNGEE, diseñado con una tensión vertical, formas abiertas y una apariencia neutral pero amigable. Fue optimizado para las interfaces de impresión, web y móvil, y tiene excelentes características de legibilidad en sus formas de letras.

Para obtener más información sobre las opciones de color con el tipo, consulte nuestra página de combo de colores
